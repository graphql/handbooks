title: Access
---

GraphQL has a lot of properties. For details of who has and getting access to
these, please see the
[Private GraphQL TSC Handbook Extensions](https://github.com/graphql/tsc/blob/main/HANDBOOK.md).

- GitHub - TSC members should be admins
- npm `graphql`, `@graphql`, `@graphiql` - TSC members may request access
- jsr.io
- YouTube: @GraphQLFoundation (working group meetings), @GraphQLTV (conference
  and locals talks), @GraphQLFoundationFR (French language talks)
- Zoom (for working group recordings)
- [Discord](https://discord.graphql.org)
- Reddit r/graphql
- Typefully.com (for posting to BlueSky, LinkedIn, Twitter/X and other
  platforms)
- [Bluesky @graphql.org](https://bsky.app/profile/graphql.org)
- LinkedIn
  [GraphQL Foundation](https://www.linkedin.com/company/graphql-foundation/)
- [Twitter/X @graphql](https://twitter.com/graphql)
- AdSense
- Meetup.com
- Netlify
- groups.io mailing lists
- [Foundation calendar](https://calendar.graphql.org)
- Cloudflare
- [Crowdfunding](https://crowdfunding.lfx.linuxfoundation.org/projects/graphql)
- Domain registration and DNS nameserver configuration via LF
- Reimbursement processes via LF
